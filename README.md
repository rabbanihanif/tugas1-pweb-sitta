# Sistem Informasi Bahan Ajar – PBW Tugas 1 (Universitas Terbuka)

Aplikasi web sederhana berbasis **HTML, CSS, dan JavaScript DOM** untuk simulasi login, data stok bahan ajar, dan tracking pengiriman.

## Fitur
- Login dengan validasi form
- Tabel data stok bahan ajar (dataBahanAjar)
- Pencarian nomor DO pada halaman tracking
- Tema dark mode modern

## Struktur
- `index.html` – halaman login
- `stok.html` – data bahan ajar
- `tracking.html` – pelacakan pengiriman
- `js/data.js` – sumber data pengguna, stok, dan pengiriman
- `css/style.css` – desain tampilan
